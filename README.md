## Abandoned

This project is currently abandoned in favour of [TellyFin](https://github.com/AlenGeoAlex/TellyFin). Although both of them are for personal use and not for public use, `arr-downloader` still had to manually do a lot of things.

Tellyfin is capable in much more terms for me, Tellyfin can automatically take care of downloading moveies/series from Telegram, or decrypting/decoding DLC's and grab the link etc.
Tellyfin is also capable of inteligently infer the name of the series movie with use of LLM which helps in automatically creating the folders etc and placing it into my Jellyfin server.

So, good bye my old friend. You will not be that missed :), JK... genuinly, this project helped me initially in lot of terms to organize my media server initially.
